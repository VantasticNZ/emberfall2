# PARKED 3D KITS (not for the 2D pipeline)

Emberfall 2 is a **2D** game. These 3D kits are all **CC0 / AI-safe** and worth
keeping on record, but they are **not** used by the current pipeline.

**They are catalogued here but the binaries are NOT copied into the repo.** The
kits total well over 1 GB (one pack, "Modular …", is ~294 MB); committing them
would bloat git history and every clone/push. They stay in
`C:\Users\Administrator\Downloads`, licence-vetted (see `../_INDEX.md`). If 3D is
ever revived, move the wanted kits here — or, better, into an external assets
store / Git LFS — rather than into normal git history.

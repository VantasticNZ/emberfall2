# EMBERFALL — COHERENCE REVIEW (whole-design end-to-end check)

> Read the full design (Quality Bible, World Map, Story spine, Main flesh M1-M20, Side flesh,
> Systems) end-to-end. Checks: do threads connect? do choices pay off? does the map hold the story?
> contradictions? gaps? Per the PLAN's coherence gate before BUILD.

================================================================
## 1. THE SPINE — IS IT WHOLE? ✓ (mostly)
================================================================
Setup (god/lie) -> childhood bond -> catastrophe -> 4 region truths each cracking the lie -> Sela
betrayal point -> Spire + 3 endings. The mystery escalates logically: M10 (it's alive) -> M12 (the
order knew) -> M14 (a betrayal hid a shard) -> M16 (the full horror) -> M17 (Sela's design) -> M19
(full truth). Each region adds ONE turn of the screw. COHERENT.
GAP FIXED: ensure each region's TRUTH is gated to its dungeon so players can't get M16's horror before
M10's basics — enforce via main-quest order (Marsh first; others soft-gated by tools).

================================================================
## 2. DEED-MEMORY PAYOFFS — DO CHOICES ECHO? ✓
================================================================
Seeded -> paid off:
- Chicken (M2) -> adult Greenhollow callback + Spire cameo (M18) + epilogue (M20). ✓
- Coin (M3) -> merchant attitude/prices as adult. ✓
- Secret/cave (M4) -> carving foreshadows god; cave revisited adult. ✓
- M10 Hagga choice -> ally strength + Sela's M17 betrayal intensity + Liberator gating. ✓
- M16 mercy/power -> Purity -> Tyrant/Liberator gating + epilogue. ✓
- E2 settlement, P3 miner, T1 smuggler -> faction rep -> NPC reactions + epilogue cards. ✓
- T5 betrayal seeded by earlier deed -> vengeance/mercy -> Morality. ✓
- G4 Mara, G7 orchard kid -> recurring NPC states. ✓
EVERY major choice has a downstream payoff. COHERENT. 
GAP FIXED: the deed-memory system (QUALITY-BIBLE Part 1) must be built FIRST in Phase 2 so all these
hooks have somewhere to write — flagged as build-order priority #1.

================================================================
## 3. MAP HOLDS THE STORY? ✓
================================================================
Greenhollow hub (childhood+return+hub), 4 spoke regions each with town+dungeon+tool+shard+truth+boss,
Spire endgame. Tools gate progression + enable backtrack (lantern/grapple/hookshot/fire-frost). The
Spire visible throughout = orientation. Each region's side quests live in that region. COHERENT.
GAP FIXED: tool-gating order must be enforced so you can't reach the Spire (M18) without all 4 tools —
the ascent gauntlet (M18) requires each tool, which naturally enforces it. ✓

================================================================
## 4. ENDINGS GATED CONSISTENTLY? ✓
================================================================
- WARDEN: default, most players. ✓
- TYRANT: CRUEL + CORRUPT (from M16 power, T5 vengeance, cruel deeds). ✓
- LIBERATOR: PURE + truths-gathered + M16 mercy + M10 Hagga-allied. ✓
Each gated by accumulated Morality/Purity/specific deeds — all of which are LOGGED by named quests
above. The gating inputs all EXIST in the design. COHERENT.
GAP FIXED: need a visible-enough karma/purity readout (QUALITY-BIBLE) so players sense which ending
they're trending toward without it being a checklist — Status display + NPC reactions handle this. ✓

================================================================
## 5. TONE MIX PRESENT THROUGHOUT? ✓
================================================================
Wholesome (childhood, rebuilding, lighthouse), funny/rude (Fatley, frog, tavern songs, gestures,
funny names, "suck it"), dark (catastrophe, the god's agony, betrayals), heartfelt (Bram, Weeping
Tree, the choice). Spread across all acts, not clustered. COHERENT. Funny names on side chars only;
main cast (Bram/Hagga/Sela) stay serious. ✓

================================================================
## 6. SYSTEMS SUPPORT THE CONTENT? ✓
================================================================
Quests reward weapons/armour/trinkets/spells/abilities that exist in Systems. Tools (lantern etc.)
are both quest rewards AND Metroidvania keys AND combat/puzzle. Skill tree unlocks (see-markers via
Fatley, dash, spin-hit) tie to quests. Economy sinks (repair, oil, property) have income sources
(jobs, bounties). Difficulty setting scales combat. COHERENT.
GAP FIXED: the "see-quest-markers" unlock requires Fatley (G1) done FIRST — ensure G1 is available
early in Greenhollow return (M7). ✓

================================================================
## 7. PACING / ORDER ✓
================================================================
~1hr childhood -> catastrophe -> Marsh (intro region, easiest) -> Peaks/Coast (mid, tools stack) ->
Emberwood (heaviest themes, pre-climax) -> Spire (climax). Difficulty + theme escalate together.
Side quests + jobs pad playtime toward ~30hr WITHOUT filler (quality-gated). COHERENT.
NOTE: ~30hr depends on side-quest VOLUME — the named ~30 + jobs are the core; more quality side
quests can be added as data to reach 30hr. Quality-over-quantity rule holds.

================================================================

================================================================

memory; the map holds the story; endings are gated by inputs that all exist; tone mix is spread;
systems support the content; pacing escalates. Gaps are minor + scheduled, not structural.
DESIGN PHASE = COMPLETE + COHERENT. Ready to sign off and move to PHASE 2 BUILD (region-by-region,
deed-memory + core systems first, then Greenhollow + Marsh as the first shippable slice).

# EMBERFALL 2 — CREDITS + HOW TO SUPPORT THE CREATORS

> The people whose art + audio make this game. **Generated/maintained from
> [ASSET-LEDGER.md](ASSET-LEDGER.md) per the Attribution + Support Policy** — every shipped
> asset records the creator, source, licence, and a **support/koha link** where one exists.
> This doc is the source for an in-game **credits screen** and for **paying the creators back**.
>
> Licence shorthand: **CC0** = public domain (no credit required — we credit anyway). **OGA-BY**,
> **CC-BY**, **CC-BY-SA**, **GPL** = credit required; CC-BY-SA / GPL are copyleft (a commercial
> ship keeps the same licence + this credits screen). ⭐ = a verified Patreon/donate page (best
> koha target); other links are the creator's canonical **OpenGameArt profile** (contact/support).
> "(verify handle)" = the OGA profile follows the standard `opengameart.org/users/<handle>`
> pattern but wasn't individually confirmed this pass — confirm before a commercial ship.
